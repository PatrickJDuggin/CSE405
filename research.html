So far I know that CGI in not a coding language but rather a set of rules in regards to website programing. It is primarily used in regards to running the program within the website. This can be done with many different programs but in regards to this project I will be using c++.

So what are the standards in regard to CGI? Here are a few basic ones

literal ""

or |

Param or ()

pointer or *

[]

iterator n

alpha/alphabet, lowe4r = lowercase, higher = big case

digit numbers

alphanumeric, a combination of the two

The following are envioronment variables, used in CGI

AUTH_TYPE - authroization type, determines access

CONTENT_LENGTHf explanatory, it is the length of the content

CONTENT_TYPE - the v type of the content

GATEWAY_INTERFACE - a help page for the gatewasy ,from what I gathered

HTTP_ACCEPT - the type of the accepted data

HTTP_USER_AGENT the program used to access, like a browser for example

PATH_INFO - the usage of the path

PATH_TRANSLATED - gives literal path

QUERY_STRING - checks the strings value

REMOTE_ADDR - gives IP addresss

REMOTE_HOST - hostname of the client

REMOTE_IDENT - finds ID

REMOTE_USER - finds user info

REQUEST_METHOD checks http request type

SCRIPT_NAME finds virtual path

SERVER_NAME - gives the official name of the server www.*.com for example

SERVER_PORT - The port that accesses the server

SERVER_PROTOCOL - the protocol the server uses

SERVER_SOFTWARE - the of the software used to run the server
